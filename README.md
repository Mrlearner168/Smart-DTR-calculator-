Helps calculate time and a day . It also shows continous time insertion from time in (ex. 8:00) to time out (ex. 17:00) . This is a 24 time format . 
If you mistakenly input wrong time in and out result , you can type  s 30m , s stands for subtract and 30m is minutes depends on the wrong input of hrs. Other example if hrs is 1h and 30m . it will subtract 1 hour and 30 minutes 
Make sure all data is correct followed by 24h HH:MM format.
To run make sure you already install python 
For stop type "q" and it will quit.
Now enjoy a terminal based DTR calculator for easy time calculation
